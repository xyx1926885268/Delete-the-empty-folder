# 空文件夹删除器

#### 介绍
还在为空文件夹太多一个一个删除费时间而烦恼吗？快来试试这款应用吧！

#### 软件架构

基于python3.9.6（python3.x皆可）以及os,threading和datetime库编写


#### 安装教程

1.  安装python3.9.6，双击空文件夹删除器执行.py文件
3.  输入想扫描的文件夹（如：D:\）,如何按回车键执行（经测试扫描系统盘时间会长）
3.  或者直接下载空文件夹删除器.exe双击运行
